# INFO Aufbau Reports 


### BB_[Version]_[Level]\_[Vergleichsfunktion]\_[Übereinstimmung]

##### Version (siehe Readme im Hauptordner)
* V1 = erste Version 
* V2 = zweite Version 

##### Level 
* VAR = Variablenlevel
* PATH = Pfadlevel
* FUN = Funktionslevel  

##### Vergleichsfunktion
* HAM = Hamming 
* EU = Euklidisch 
* LSH = LSH + Hamming 
* LSH_BC = LSH + Hamming + BucketCleanup 

##### Übereinstimmung 
* 100% - Threshold
